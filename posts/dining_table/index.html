Dining Table

Dining table I made for my new house.

I needed a dining table for the house I was buying, so of course I decided to make one. I used boards leftover from previous projects, which drove the table design and dimensions. This project used up all the large boards I had lying around, and I didn’t have to buy any new material, which I thought was pretty incredible.
