Cask 3D Dose

  2014-06-21

Rendering of dry storage cask surface dose rates.

Map of BWR reactor core showing relative pin powers on color scale. Generated with desktop application I wrote in C++ with Qt that reads proprietary data files and allows user to interactively view axial and radial maps of the data arrays (power, burnup, etc).
